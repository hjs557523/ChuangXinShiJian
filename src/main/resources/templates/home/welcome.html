欢迎使用教师端web后台管理系统

  • 

    项目介绍

    • 学生端：微信小程序
    • 教师端：web管理后台

    1、基于github项目进行团队创建，实现创新实践小组的团队管理，能够创建团队，添加团队成员等团队操作

    2、任务分工管理，创建任务，设置任务截止时间，指定任务执行人，订阅消息通知，并且和github上的项目进行同步

    3、基于github API，对小组项目的进展情况进行追踪，实现对每位成员的工作统计，以图表化的形式进行展示

    4、基于github API，实现小组成员以任务issue为中心而展开的留言和探讨

    5、实现对每位成员的周报管理，完成对每周汇报的填写和提交

  • 

    技术框架

    运行系统环境：Windows / Linux

    开发工具：IntelliJ IDEA / Visual Studio Code

    应用服务容器：tomcat 7.0

    数据库版本：MySQL 5.7

    Redis缓存版本：redis 3.2.100

    后端技术框架：Spring Boot + Mybatis + Shiro + JDK 1.8

    web端技术框架：LayUI + Vue.js + Echarts + Thymeleaf

    依赖包管理：Maven 2.0

    其他：HttpClient + WebSocket + MyBatis Generator + PageHelper + Log4j + Alibaba Druid
